## ❤️ Heart Disease Prediction — Model Comparison Analysis

A machine learning project built using Python and Jupyter Notebook that compares multiple classification models for predicting heart disease based on medical and patient health data. The project demonstrates data preprocessing, exploratory analysis, model training, evaluation, and performance comparison using real-world healthcare datasets.

## 🚀 Features
* **Heart Disease Prediction System** – Predict the likelihood of heart disease using ML models.
* **Multiple Model Comparison** – Compare classification algorithms and their performance.
* **Healthcare Data Analysis** – Analyze medical and patient health attributes.
* **Data Cleaning & Preprocessing** – Prepare healthcare dataset for machine learning.
* **Exploratory Data Analysis (EDA)** – Visualize patterns and medical trends.
* **Feature Correlation Analysis** – Identify influential health-related variables.
* **Model Training & Evaluation** – Train and evaluate classification algorithms.
* **Performance Metrics Comparison** – Compare accuracy, precision, recall, and F1-score.
* **Visualization & Insights** – Generate analytical graphs and comparison plots.
* **Real-World Medical Dataset** – Practice healthcare-focused predictive analytics.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Model-Comparison.git
```
2. **Navigate to the project folder**:
```
cd Model-Comparison
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
heart_model_comparison.ipynb
```
